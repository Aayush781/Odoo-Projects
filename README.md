# Odoo-Projects
Odoo projects 


1)Asset Management(maintenace_assets)- Extends Odoo's native Maintenance module into a full Asset Management System. Assets are classified by Asset Type and Asset Category, assigned to employees and Asset teams, and access is controlled per team.

Features--
**Asset Type Dashboard** — Kanban view showing all asset types. Each card links to the categories under that type.                                                               
**Asset Classification** — Two-level hierarchy: Asset Type → Asset Category → Asset      (Equipment).                                                                                 
**Team-Based Access Control** — Users only see assets belonging to their maintenance       team. Admins and Equipment Managers see all.                                                
**Asset Assignment Email** — Automatically sends a notification email to an employee       when an asset is assigned to them.                                                          
**Asset Analysis Report** — Graph and pivot views showing asset distribution by type       
and category.                                                                               
**Menu Restrictions** — Non-admin users only see the Asset Management menu; all othe       
apps are hidden.
